# Oil-War-and-Wall-Street-Impact-of-the-Russia-Ukraine-Conflict-on-Brent-Crude-and-the-S-P-500
Time-series econometric analysis examining how the Russia–Ukraine war altered the Brent crude oil–S&amp;P 500 relationship, revealing structural breaks, volatility instability, and limits of conventional GARCH modelling during geopolitical crises.
